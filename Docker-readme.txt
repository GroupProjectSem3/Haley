To Run the application through docker on local and further deploy it to Kubernetes follow th steps

1- Download docker desktop
2- Enable all the HTTP/TCP protocols
3- Run docker deamon on desktop
4- Navigate to the webapp
5- Make virtual environmnet
6- Activate the environment
7- Run the command - docker-compose run web django-admin startproject app .
8- RUN the command - docker-compose up
9- open browser nad navigate to http://localhost:8000/
